# romview

## overview
ROM image viewer for HSP3.

## 特徴

・ヘッダー情報の確認。

・サムチェック。

## 対応する拡張子

nes : famicom

smc : super famicom

gb : gameboy

ws/wsc : wonderswan

p6/cas : PC-6001 cassette tape

bin : megadrive or MSX

rom : MSX

pce : PC Engine
